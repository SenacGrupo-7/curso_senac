No CSS nós possuimos os seletores por ID

  • Seletores por Tipo: Buscam elementos por uma tag HTML específica.
  • Seletores por ID: Busca os elementos através ID.
  • Seletores por Classe: Buscam elementos através do atributo "class".
  • Seletor Universal: Seleciona todos os elementos HTML.
  • seletor de Atributos ([atrib]):Seliciona elementos que possuem um atributo específico em nosso. documento HTML e conseguimos te=ambém buscar com um valor específico.
