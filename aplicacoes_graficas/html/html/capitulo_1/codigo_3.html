📌 Explicação do Código


HTML básico:
    - Define a estrutura da página com <html>, <head>, e <body>.
    - Inclui um canvas (<canvas id="canvas">) onde o desenho será feito.

Script JavaScript (desenhar_canvas):
    - Obtém o elemento <canvas> pelo ID (document.getElementById("canvas")).
    - Verifica se o navegador suporta getContext("2d").
    - Desenha uma elipse laranja usando ctx.ellipse(x, y, raioX, raioY, rotação, início, fim).
    - Adiciona o texto "senac" e "são paulo" com ctx.fillText().
    - Finaliza com ctx.stroke() para exibir os traços.

onload="desenhar_canvas();" no <body>:
    - Garante que o script só seja executado após a página carregar.

🚀 O que esse código faz?
    - Cria um logotipo estilizado do SENAC usando a API Canvas.
    - Desenha uma elipse inclinada e coloca os textos "senac" e "são paulo" sobre ela.
